Module 7: Streaming Data with RedPanda and PyFlink,
Tech Stack

    RedPanda : A modern, Kafka-compatible event streaming platform built in C++. It is designed to be a faster, lighter alternative to Kafka that eliminates the need for JVM (Java) and ZooKeeper, making it much easier to deploy and manage in Docker environments.
    PyFlink : PyFlink is the Python API for Apache Flink, a powerful framework for stateful computations over data streams. It allows you to build highly scalable, fault-tolerant streaming and batch processing applications using Python instead of Java or Scala.
